Contador de Calorias - 2º Semestre

O projeto da FATEC Jacareí propõe desenvolver uma aplicação web para contador calórico com sugestões de dietas personalizadas, usando metodologias ágeis (Scrum). A interface será feita com React, TypeScript, HTML5, CSS3 e design no Figma. Dados serão armazenados em PostgreSQL, com controle via Git/GitHub. A aplicação visa monitorar calorias e recomendar dietas conforme as necessidades dos usuários.

Tecnologias:

HTML, CSS, JavaScript, React, PostgreSQL
